# Solar Challenge Week 1

## Project Overview
This project is part of the AI Mastery training. It implements a modular Python package for solar energy analysis with a structured approach including data handling, modeling, plotting, and testing.

## Folder Structure

- `src/` : Python modules and main scripts
- `data/` : Input datasets (CSV, Excel, etc.)
- `notebooks/` : Jupyter notebooks for exploratory data analysis
- `reports/` : Generated plots, figures, and result reports
- `tests/` : Unit tests for modules
- `.github/workflows/` : CI workflows (testing, linting)
- `requirements.txt` : Python dependencies
- `.gitignore` : Files/folders to ignore in Git

## Usage
1. Create a Python virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```
2. Install dependencies:
```bash
pip install -r requirements.txt
```
3. Run main script:
```bash
python src/main.py
```
