# Solar Challenge Week 1

This repository contains the setup for the Solar Challenge Week 1 project.

## Environment Setup

Follow these steps to reproduce the Python environment:

### 1. Clone the repository
```bash
git clone https://github.com/YeabisraW/solar-challenge-week1.git
cd solar-challenge-week1

### 2. Create a virtual environment

Using Python's built-in venv:
python -m venv venv

###3. Activate the virtual environment

On Windows (Git Bash):
source venv/Scripts/activate


###4. Install dependencies
pip install -r requirements.txt

###5. Verify installation
python --version
pip list
